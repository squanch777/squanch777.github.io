# 小结

​ 《哲学的故事》按顺序讲了关于帕拉图、亚里士多德、苏格拉底、弗朗西斯培根（有所耳闻）、斯宾诺莎（读时才第一次听说这位哲学家）、伏尔泰。

​ 通过这本书，我发现对于上述几位哲学家而言，哲学不仅仅是思考人生、思考生活，也会去寻找科学，寻找科学的实验、方法、理论、逻辑；而不是如我们寻常所听闻的西哲大部分思考于唯心、唯物；宗教；寻求真理等等思想论。

​ 古希腊提出伦理学、政治学、心理学、科学、逻辑学、形而上学等学科，并且通过观察、实验亦或者是通过构建 “乌托邦” 去提出自己关于哲学王国的思考。虽然局限于时代、技术、社会环境等因素，但是仍给予了我极大的思考空间。

​ 宗教改革、文艺复兴、资产阶级革命开始的哲学，开始着重于宗教、神的思考；科学体系的建立。培根、伏尔泰都尝试过通过著书来表明自己对科学体系的想法。

# 一句话

​ 安宁的奥秘：不强求我们的成就就与我们的欲望相等，而是降低我们的欲望使之与我们的成就想持平。

# 自己的想法

​ 对于伦理学、逻辑学、犬儒主义是我想进一步了解的。

​ 他们所讲的伦理学的伦理并非是我们传统所谓的伦理，所以我需要主动去了解去思考。

​ 逻辑学能够帮我更好地理清思路，思考处事的原则与方式。

​ 至于犬儒主义，我在知乎上看到过一种观点，不是绝对的 躺平就不是犬儒主义。如果你做不到平淡的看待一切就不是真正的犬儒主义。

# 彩蛋

​ 伏尔泰的父亲说过：想无益于社会、连累亲戚、打算饿死的人才干文学这一行。

​ 同时，令我没想到的是伏尔泰作为一个哲学家、政治家之余还是一位很厉害很厉害的戏剧家且年少成名，创作时间之长。
