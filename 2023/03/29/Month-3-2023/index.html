# 三月小结

# 今天跟宝峰哥聊天，宝峰哥那时候可以在百度工作较为稳定还有关系可以稳定生活，但是放弃选择出去创业，然后失败欠债现在在我爸这里一起工作。我很想问后悔嘛？

我不知道我以后会不会后悔，虽然回首过去，于我而言肯定有更好的选择比如更努力，选个其他专业，有恒心与决心。但是木已成舟，就无须考虑过去，从现在来看不管怎么做都不算是晚。

PP 姐那时候问我从中汇离开去致同后悔吗？也许是的，再从致同跑了再到如今的高不成低不就那又如何。

他人悔之我不悔！

# 工作

​ 考公失利，但也是自己不够努力，差 0.7 分，着实可惜

​ 投递了杭州的财务工作，柯桥确实没有工作

​ 实习中信证券，很难转正，销售很好听叫市场营销

# 思考

个人缺点：这次考公失利，以及工作的抉择下，我是一个瞻前顾后，能力有限，眼界还不够宽阔；

没有绝对的恒心，似乎除了游戏？！

现在阶段没什么上进心

个人优点：还是很拿得起放得下

目标：不知道为何，非常想出国定居，考虑自身情况，美加澳太贵，也只有留学一条途径，想读 CS 硕，跨专业申请难度。

要有决心
